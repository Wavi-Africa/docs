Wavi is the single place to send SMS, Email and OTP to customers across Africa. One API, one dashboard, one wallet.

## What's inside

- **Quickstart** - install, first SMS, first email, first OTP
- **SMS** - direct carrier routes, sender IDs, delivery receipts
- **Email** - managed SMTP, DKIM and SPF, templates, webhooks
- **OTP** - one endpoint, channel picked for you
- **WhatsApp** - coming soon, join the waitlist
- **API Reference** - every endpoint, auto-generated from the API
